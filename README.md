<li>Clone this repo:

```$xslt
git clone https://github.com/Adi1222/Portfolio.git
```

<li>Install the dependencies:

```$xslt
npm install 
# or
yarn install
```

<li>Run the development server:

```
npm start
```

Open [http://localhost:3000](http://localhost:3000) on your browser to see the result.
